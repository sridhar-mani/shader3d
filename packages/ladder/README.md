# @shader3d/ladder

Progressive learning system and CLI for Shader3D.

Provides:
- Skill level detection
- Project scaffolding
- Magic comments (Level 0)
- CLI tools

## Install

```bash
npm install @shader3d/ladder -D
```

## CLI

```bash
# Create new project
npx shader3d init my-project

# Analyze code level
npx shader3d upgrade

# Build shaders
npx shader3d build

# Watch mode
npx shader3d watch
```
